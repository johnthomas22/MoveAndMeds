# MoveAndMeds
An app for recovery from surgery, possibly other hospitalisations. 

Intelligently scans the exercises from hospital physio sheets, setting up default schedules. Notifications then remind the user when an exercise is due.

Use this free app at your own risk. You must follow the directions of your medical team and no warranty is implied that the app will always notify you at the correct time. If you do not accept this risk, please delete the app.
